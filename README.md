## 📌 학습 내용

1. **HTML**: 문서 구조, 시멘틱 태그  
2. **CSS**: 스타일링, 박스 모델, Flexbox  
3. **JavaScript**: 기본 문법, 이벤트 처리  
4. **기본 프로젝트**: 간단한 웹 페이지 만들기

- 학습 후 직접 코드를 작성해보며 실습하는 것이 중요합니다.
